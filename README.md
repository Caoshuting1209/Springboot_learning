# Springboot_learning
Springboot learning
## shotcut

```bash
ctrl + u: #delete line
ctrl + e: #to end
ctrl + a: #to ahead
ctrl + c: #cancel command
ctrl + d: #exit(drop)

#编辑
nvim #edit
i #insert
esc -> :w #sava
esc -> :q #quit
esc -> :q! #quit without saving
esc -> dd #delete current line
esc -> o #insert a line below current line
esc -> O #insert a line above current line
esc -> gg #jump to the first line
esc -> G #jump to the last line
esc -> 0 #to ahead of the current line
esc -> $ #to end of the current line


cd #root
cd . #current
cd .. #last
mv old_path new_path #move
cp old_path new_path #copy
rm file #delete file
rm -r document #delete directory
tab
ls #list
pwd #present working directory
up arrow #last command
```
